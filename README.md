# Sistema de Sorteio – Delphi + Firebird

Projeto desenvolvido em **Delphi** utilizando **FireDAC** e **Firebird**, com foco em **boas práticas de arquitetura**, separação de responsabilidades e código organizado em camadas.

O sistema simula um **cadastro de clientes, carros e vendas**, com regras de negócio aplicadas via camada de *Service*.
